# Examples - Blueprint V2

Contoh output dari workflow Blueprint V2.

---

## Files

### 1. sample-script-output.json
Contoh output dari **The Director** (Gemini 1.5 Pro).

**Contains:**
- Video title
- 5 scenes dengan timing breakdown
- Voiceover scripts
- Visual prompts untuk setiap scene
- Sound effect prompts
- Character information
- Metadata (hashtags, CTA, target audience)

**Use case:**
- Understand struktur script yang dihasilkan
- Learn how to write good visual prompts
- Template untuk manual script writing

---

### 2. sample-character-sheet.json
Contoh character definition untuk "Alex Chen" protagonist.

**Contains:**
- Complete character description
- Physical attributes detail
- Clothing and accessories
- Consistency keywords
- Prompt templates
- Usage notes

**Use case:**
- Template untuk membuat character baru
- Reference untuk consistency guidelines
- Example prompts untuk character generation
